To compile and execute lexical analyzer, follow following step:

1. make result_1 (it will generate myparserParser.java, myparserLexer and myparser.tokens)
2. make result_2 (it will compile all .java files, generate relative .class files)
3. make reulst_test_1 (it will parsing test.c)
4. make result_test_2 (it will parsing test2.c)
5. make result_test_3 (it will parsing test3.c)

Using antlr-3.5.3-complete-no-st3.jar.

NOTE: In Makefile, CLASSPATH need to change to your own directory of antlr.jar!
